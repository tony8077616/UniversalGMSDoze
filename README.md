# Universal GMS Doze
This module enables optimized Google Play services. [More details in support thread.](https://forum.xda-developers.com/apps/magisk/module-universal-gms-doze-t3853710)

## Compatibility
- Android Marshmallow+
- SELinux Enforcing
- All root solutions
